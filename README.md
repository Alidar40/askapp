# AskApp

## Описание
Это сервис вопросов и ответов, в котором пользователи могут задавать друг другу вопросы и получать ответы.

### Стек
- Бэкенд Django
- База данных MySQL
- nginx для отдачи статических файлов
- Фронтенд: JS, Django Templates, TwitterBootstrap
